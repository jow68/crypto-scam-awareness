
# Crypto Scam Awareness (Bilingual FR/EN)

Fichiers prêts pour **GitHub Pages**.

## Déploiement
1. Créez un dépôt public `crypto-scam-awareness`.
2. Téléversez le contenu de ce dossier (`index.html`, `assets/...`).
3. `Settings → Pages` → Source : `Deploy from a branch` (main / root).
4. Votre URL : `https://<your-username>.github.io/crypto-scam-awareness/`.
5. Placez vos infographies dans `assets/images/` (`infographie1.png`, `infographie2.png`).
6. Les boutons de partage utilisent automatiquement l’URL de GitHub Pages ci-dessus.
